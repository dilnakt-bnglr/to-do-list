# To‑Do List

A basic to‑do list application..  
Users can add, mark complete, update task and remove tasks

Github URL - https://github.com/dilnakt-bnglr/to-do-list

1. **Clone the repo**

   git clone https://github.com/dilnakt-bnglr/to-do-list.git

2. **Install dependencies**

   npm install # or yarn install

3. **Run the application**

   npm run dev

   The server will typically start at `http://localhost:5173/`

## Features

- Create new to‑do items.
- View a list of existing items.
- Mark items as completed.
- Delete items.
- update to-do task
